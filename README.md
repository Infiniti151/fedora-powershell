# fedora-powershell
A simple Bash script for installing/updating Powershell 7+ on Fedora as it's not available in the official repos.

## Install Powershell
```
git clone https://github.com/Infiniti151/fedora-powershell.git
cd fedora-powershell
chmod +x ./powershell.sh
./powershell.sh
```

## Update Powershell
Run the script again to update Powershell
